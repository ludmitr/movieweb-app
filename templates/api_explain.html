RESTful API Documentation

Manage Users (POST, GET)

Endpoint: /api/users

GET:

Returns a JSON array of all users.

POST:

Adds a new user with the provided user name in the JSON payload:

{
    'user_name': 'string'  // Required
}

Delete User by ID (DELETE)

Endpoint: /api/users/<user_id>

Deletes the user with the provided ID.

Get User Movies (GET)

Endpoint: /api/users/<user_id>/movies

Retrieves movies associated with a specific user.

Add Movie to User (POST)

Endpoint: /api/users/<user_id>/movies

Adds a movie to a user's collection using the provided title in the JSON payload:

{
    'title': 'movie_name'  // Required
}

Delete Movie of User (DELETE)

Endpoint: /api/users/<user_id>/movies/<movie_id>

Deletes the specified movie from the specified user's collection.

Review Manager (POST, DELETE)

Endpoint: /api/users/<user_id>/movies/<movie_id>/review

DELETE:

Deletes the review for a given movie and user.

POST:

Adds a review to a movie for a specific user, with the provided review in the JSON payload:

{
    'review': 'text'  // Required
}

Get All Reviews for Movie (GET)

Endpoint: /api/movies/<movie_id>/reviews

Retrieves all reviews for a given movie by its ID.
